## Configuration files.

Various confguratin files either push/mounted to a target file inside containers (utilising configs primitive) or pushed into environment variables inside containers.


1. conf/config.yaml         -   Flink Jobmanager/Taskmanager Configuration 

2. pg_hba.conf              -   PostgreSql

3. postgresql.conf          -   PostgreSql

4. log4j-*.properties       -   Log4j configuration files

## Configuration files.

Various confguratin files either push/mounted to a target file inside containers (utilising configs primitive) or pushed into environment variables inside containers.


1. flink/config.yaml            -   Flink Jobmanager/Taskmanager Configuration 

2. log4j/log4j-*.properties     -   Log4j configuration files

3. pg/pg_hba.conf               -   PostgreSql

4. pg/postgresql.conf           -   PostgreSql
